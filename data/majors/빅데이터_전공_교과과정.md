# 빅데이터 (Big Data) 전공

## 전공 개요

빅데이터의 접목을 통하여 경영학, 인문사회학, 의료 및 보건학, 공학, 농학, 자연과학 등 거의 모든 분야에서 미래사회에 적합한 인재 양성이 가능한 시대가 되었다.

본 연계전공은 다양한 분야의 학생들에게 빅데이터 역량을 교육하여 미래사회를 주도할 인재양성을 목표로 한다.

## 전공간 중복 학점인정 교과목

### 📌 제도 설명
**각 개설학과 소속 학생이 해당학과 교과목을 수강하는 경우 중복인정함**

**핵심 개념**
1. **이중 학점 인정**: 본전공 소속 학생이 해당 학과 교과목을 수강하면, 본전공과 빅데이터 전공 학점 모두 인정
2. **중복 수강 방지**: 같은 내용을 두 번 들을 필요 없음
3. **효율적 학점 관리**: 융합전공 이수 부담 경감

**활용 예시**
```
예시 1: 소프트웨어학부 학생 + 빅데이터 전공 이수 중
→ 소프트웨어학부 "객체지향프로그래밍(5118009)" 수강
→ 소프트웨어학부 전공 학점 인정 ✅
→ 빅데이터 전공 전공선택 학점 인정 ✅
→ 한 과목으로 두 전공 동시 충족!

예시 2: 경영정보학과 학생 + 빅데이터 전공 이수 중
→ 경영정보학과 "데이터마이닝(6208071)" 수강 (필수 과목)
→ 경영정보학과 전공 학점 인정 ✅
→ 빅데이터 전공 전공선택 학점 인정 ✅
→ 필수 과목 이수와 융합전공 학점 동시 획득!

예시 3: 정보통계학과 학생 + 빅데이터 전공 이수 중
→ 정보통계학과 "데이터마이닝(5961318)" 수강
→ 정보통계학과 전공 학점 인정 ✅
→ 빅데이터 전공 전공선택 학점 인정 ✅
→ 타학과 인정 교과목 리스트에 있으므로 중복 인정!
```

**💡 적용 원칙**: 위의 "타학과(부, 전공) 전공선택 인정 교과목" 목록에 있는 모든 과목들은 각 개설학과 소속 학생이 수강할 경우 본전공과 빅데이터 전공 학점으로 중복 인정됩니다.
