## 전공간 중복 학점인정 교과목

### 📌 제도 설명
**각 개설학과 소속 학생이 소속학과 교과목을 수강하는 경우 9학점까지 중복 인정함**

**핵심 개념**
1. **이중 학점 인정 (제한적)**: 본전공 소속 학생이 해당 학과 교과목을 수강하면, 본전공과 벤처비즈니스 전공 학점 모두 인정
2. **9학점 제한**: 다른 융합전공과 달리 **최대 9학점까지만** 중복 인정 가능
3. **중복 수강 방지**: 같은 내용을 두 번 들을 필요 없음
4. **효율적 학점 관리**: 융합전공 이수 부담 경감 (단, 제한적)

### ⚠️ 중요: 9학점 제한

```
벤처비즈니스 전공의 특별한 규정:
다른 융합전공들은 중복 인정 학점에 제한이 없지만,
벤처비즈니스 전공은 최대 9학점까지만 중복 인정됩니다!

예시:
경영학부 학생이 벤처비즈니스 융합전공 이수 중
→ 경영학부 교과목 중 3과목(각 3학점) = 9학점까지 중복 인정 ✅
→ 4과목째부터는 중복 인정 불가 ❌
→ 벤처비즈니스 전공 학점으로만 인정
```

**활용 예시**
```
예시 1: 경영학부 학생 + 벤처비즈니스 전공 이수 중
중복 인정 가능한 과목들:
1. 회계원리(6210001) - 3학점 ✅
2. 마케팅관리론(6210003) - 3학점 ✅
3. 재무관리(6210004) - 3학점 ✅
→ 총 9학점 중복 인정

4. 경영전략론(6210068) - 3학점 ❌
→ 9학점 초과, 벤처비즈니스 전공 학점으로만 인정
→ 경영학부 전공 학점으로는 별도로 계산 안됨

예시 2: 국제경영학과 학생 + 벤처비즈니스 전공 이수 중
→ 국제경영학과 "글로벌비즈니스의 이해(6209100)" 수강 - 3학점
→ 국제경영학과 "국제마케팅(6209009)" 수강 - 3학점
→ 국제경영학과 "글로벌벤처기업전략(6209114)" 수강 - 3학점
→ 총 9학점까지 국제경영학과 + 벤처비즈니스 전공 동시 인정 ✅

→ 국제경영학과 "국제비즈니스커뮤니케이션(6209090)" 수강 - 3학점
→ 9학점 초과, 벤처비즈니스 전공으로만 인정 ❌
```
