## 전공 개요

## 전공간 중복 학점인정 교과목

### 📌 제도 설명
**서로 다른 학과의 유사한 과목을 같은 과목으로 인정하여, 한 과목만 수강해도 여러 전공의 학점으로 인정해주는 제도**

**핵심 개념**
1. **과목 선택의 유연성**: 같은 내용의 과목이 여러 학과에 있을 경우, 어느 학과 것을 들어도 인정
2. **이중 학점 인정**: 본전공 소속 학생이 해당 학과 교과목을 수강하면, 본전공과 이차전지 융합전공 학점 모두 인정
3. **중복 수강 방지**: 같은 내용을 두 번 들을 필요 없음

**활용 예시**
```
예시 1: 과목 선택의 유연성
학생이 "물리화학"을 배워야 함
→ 옵션 A: 안전공학과 "물리화학(6543093)" 수강 ✅
→ 옵션 B: 화학공학과 "물리화학Ⅰ(6550009)" 수강 ✅
→ 옵션 C: 공업화학과 "물리화학Ⅰ(6552103)" 수강 ✅
→ 옵션 D: 신소재공학과 "물리화학(6538006)" 수강 ✅
→ 네 가지 중 시간표가 맞는 것을 선택하면 됨!

예시 2: 이중 학점 인정
화학공학과 학생 + 이차전지 융합전공 이수 중
→ 화학공학과 "공학수학Ⅰ(6550103)" 1개 수강
→ 화학공학과 전공 학점 인정 ✅
→ 이차전지 융합전공 "공학수학" 학점 인정 ✅
→ 한 과목으로 두 전공 동시 충족!
```
