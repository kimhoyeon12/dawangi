## 전공간 중복 학점인정 교과목

### 📌 제도 설명
**서로 다른 학과의 유사한 과목을 같은 과목으로 인정하여, 한 과목만 수강해도 여러 전공의 학점으로 인정해주는 제도**

**핵심 개념**
1. **과목 선택의 유연성**: 같은 내용의 과목이 여러 학과에 있을 경우, 어느 학과 것을 들어도 인정
2. **이중 학점 인정**: 본전공 소속 학생이 해당 학과 교과목을 수강하면, 본전공과 공공데이터사이언스 융합전공 학점 모두 인정
3. **중복 수강 방지**: 같은 내용을 두 번 들을 필요 없음

**활용 예시**
```
예시 1: 과목 선택의 유연성
학생이 "경영통계학"을 배워야 함
→ 옵션 A: 소비자학과 "연구방법 이론 및 실습(7717100)" 수강 ✅
→ 옵션 B: 경영학부 "경영통계(6210010)" 수강 ✅
→ 둘 중 시간표가 맞는 것을 선택하면 됨

예시 2: 이중 학점 인정
경영정보학과 학생 + 공공데이터사이언스 융합전공 이수 중
→ 경영정보학과 "경영정보시스템(6208004)" 1개 수강
→ 경영정보학과 전공 학점 인정 ✅
→ 공공데이터사이언스 융합전공 필수 학점 인정 ✅
→ 한 과목으로 두 전공 동시 충족!
```
