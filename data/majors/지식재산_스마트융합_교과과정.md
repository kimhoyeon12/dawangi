## 타학과(부, 전공) 전공선택 인정 교과목

### 📌 제도 설명
**다른 학과나 전공의 과목을 수강해도 지식재산 스마트융합 전공의 전공선택 학점으로 인정해주는 제도**

**목적**
- 융합전공 특성상 다양한 분야의 지식이 필요
- 관련 학과의 전문 과목을 자유롭게 수강 가능
- 학생의 관심사와 진로에 맞춘 맞춤형 학습 지원

**활용 예시**
```
지식재산 융합전공 학생이
→ 약학과의 "제약산업학(8310075)" 수강
→ 지식재산 전공의 전공선택 학점으로 인정 ✅
```

## 전공간 중복 학점인정 교과목

### 📌 제도 설명
**서로 다른 학과의 유사한 과목을 같은 과목으로 인정하여, 한 과목만 수강해도 여러 전공의 학점으로 인정해주는 제도**

**핵심 개념**
1. **과목 선택의 유연성**: 같은 내용의 과목이 여러 학과에 있을 경우, 어느 학과 것을 들어도 인정
2. **이중 학점 인정**: 본전공 소속 학생이 해당 학과 교과목을 수강하면, 본전공과 융합전공 학점 모두 인정
3. **중복 수강 방지**: 같은 내용을 두 번 들을 필요 없음

**활용 예시**
```
예시 1: 과목 선택의 유연성
학생이 "회계원리"를 배워야 함
→ 옵션 A: 경영학부 "회계원리(6210001)" 수강 ✅
→ 옵션 B: 국제경영학과 "회계원리(6209035)" 수강 ✅
→ 둘 중 시간표가 맞는 것을 선택하면 됨

예시 2: 이중 학점 인정
국제경영학과 학생 + 지식재산 융합전공 이수 중
→ 국제경영학과 "회계원리(6209035)" 1개 수강
→ 국제경영학과 전공 학점 인정 ✅
→ 지식재산 융합전공 학점 인정 ✅
→ 한 과목으로 두 전공 동시 충족!
```
