# 다왕 아바타 이미지

이 폴더에 다왕 캐릭터의 감정별 이미지를 추가해주세요.

## 필요한 이미지 파일 (4개)

1. **기본.png** - 중립 상태 (64x64px 이상)
2. **기쁨.png** - 기쁜 표정 (64x64px 이상)
3. **당황.png** - 당황한 표정 (64x64px 이상)
4. **뿌듯.png** - 뿌듯한 표정 (64x64px 이상)

## 이미지 사양

- **형식**: PNG (투명 배경 권장)
- **크기**: 64x64px ~ 256x256px
- **파일명**: 정확히 위의 이름과 일치해야 함

## 임시 플레이스홀더

이미지가 없는 경우, 임시로 아래 방법을 사용하세요:

1. https://via.placeholder.com/64x64.png 접속
2. 이미지를 다운로드하여 4개 파일로 복사
3. 파일명을 위의 이름으로 변경

또는 간단한 아이콘/이모지 이미지를 사용하세요.

## 캐릭터 디자인 가이드

- **스타일**: 귀여운 동그라미 캐릭터
- **색상**: 충북대 레드 (#B72654) 활용
- **표정**: 명확하고 친근한 느낌
